# Descrição

Sistema desenvolvido em linguagem C utilizando arquivos binários para gerenciamento de um streaming de vídeos.

O programa realiza:

* cadastro de vídeos;
* cadastro de usuários;
* listagem de registros;
* atualização de registros;
* remoção de registros;
* relatório de favoritos.

---

# Arquivos do Projeto

```text
streaming_de_videos/
│
├── comandos.txt
├── README.md
├── streaming_de_videos.c
└── streaming_de_videos.exe
```

## Descrição dos Arquivos

| Arquivo                 | Descrição                                                                        |
| ----------------------- | -------------------------------------------------------------------------------- |
| comandos.txt            | Arquivo contendo todas as entradas utilizadas para testar a execução do programa |
| README.md               | Arquivo com a descrição da atividade e informações do projeto                    |
| streaming_de_videos.c   | Código-fonte do sistema desenvolvido em linguagem C                              |
| streaming_de_videos.exe | Arquivo executável gerado após a compilação do programa                          |

---

# Arquivos Binários

O sistema utiliza:

* `videos.bin`
* `usuarios.bin`

Os arquivos são criados automaticamente durante a execução do programa.
